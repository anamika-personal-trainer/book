# Anamika PT – GitHub Pages

Detta är en komplett statisk one-page-hemsida för GitHub Pages.

## Filer
- `index.html`
- `style.css`
- `script.js`
- `assets/` med webbanpassade bilder och QR-kod

## Föreslagen live-länk
När GitHub Pages är aktiverat för repot `anamika-personal-trainer/book` blir adressen normalt:

`https://anamika-personal-trainer.github.io/book/`

Prissidan/QR-länken är:

`https://anamika-personal-trainer.github.io/book/#priser`

Kontrollera alltid att länken fungerar innan ni trycker visitkort.

## Så laddar du upp till GitHub
1. Öppna: https://github.com/anamika-personal-trainer/book
2. Klicka `Add file` → `Upload files`.
3. Dra in alla filer och mappen `assets` från denna zip.
4. Klicka `Commit changes`.
5. Gå till `Settings` → `Pages`.
6. Välj `Deploy from a branch`.
7. Branch: `main`, folder: `/root`.
8. Klicka `Save`.

Efter 1–3 minuter bör sidan vara live.

## Viktigt om QR-koden
QR-koden i `assets/qr-priser.png` går till:
`https://anamika-personal-trainer.github.io/book/#priser`

Om GitHub Pages ger en annan adress behöver QR-koden skapas om innan visitkort trycks.
