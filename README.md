## Uppdatera via VS Code

```bash
cd ~/ws/book
git pull
```

Kopiera in filerna från denna version. Kör sedan:

```bash
git add .
git commit -m "Clean up website typography and text"
git push
```

Vänta 1–3 minuter och testa sidan.
